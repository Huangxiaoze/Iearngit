图形编辑软件（快捷键）:
Ctrl + 鼠标左键：选中单一图形
Ctrl + D         删除选中图形
Ctrl + C         复制选中图形
Ctrl + V         粘贴选中图形
Ctrl + S         保存
Ctrl + Z         撤销生成，当前只针对新创建元素起作用
Ctrl + 上箭头         向上对齐
Ctrl + 下箭头         向下对齐
Ctrl + 左箭头         向左对齐
Ctrl + 右箭头         向右对齐
Ctrl + 0          向中对齐（以上下为参考）
Ctrl + 1          向中对齐（以左右为参考）
Ctrl + 5             向中对齐（矩阵对角线交点）
Alt + 鼠标左键：        旋转选中图形

Shift 创建矩形， 椭圆
